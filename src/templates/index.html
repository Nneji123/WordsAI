WordsAI

WordsAI a collection of Natural Language Processing apps that uses the WordsAI API to perform Natural Language Processing tasks.

Check out the Sentiment App or the Summarizer App or the Translation App.

You can check out the Github Repository for more information.

You can also visit the API documentation.

Features

  • Text Summarizer
  • Webpage Summarizer
  • Sentiment Analyzer
  • Autocorrect
  • Resume Parser
  • Wordcloud
  • Translate
  • Optical Character Recognition
  • Speech to Text
